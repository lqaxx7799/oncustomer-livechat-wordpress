Kết nối ứng dụng với ứng dụng OnCustomer Livechat để bắt đầu chăm sóc khách hàng của bạn trên website.

== Description ==
OnCustomer Livechat là plugin giúp tích hợp ứng dụng OnCustomer Livechat trên website WordPress của bạn, hỗ trợ bạn chăm sóc khách hàng của bạn trên chính website của mình.

== Installation ==
1. Vào màn hình ‘Plugins’ tại trang Admin, chọn ‘Add New’.
2. Tìm kiếm plugin ‘OnCustomer Livechat’ và chọn ‘Install Now’ để tải xuống.
3. Vào màn hình ‘Plugins’ tại trang Admin, tìm ‘OnCustomer Livechat’ và chọn ‘Activate’ để kích hoạt.
4. Sau khi kích hoạt plugin, tại menu bên trái màn hình, chọn ‘OnCustomer’.
5. Tại màn hình OnCustomer, nhấn ‘Tích hợp ngay’ để cài đặt OnCustomer Livechat trên trang web của mình. Nếu bạn chưa có tài khoản OnCustomer Livechat, bấm vào đường dẫn ‘Tạo tài khoản’ để đăng ký sử dụng OnCustomer Livechat.
6. Sau khi chọn ‘Tích hợp ngay’, bạn sẽ được điều hướng sang màn hình xác thực tích hợp. Xác nhận để điều hướng trở lại màn hình Admin.
7. Màn hình thông báo tích hợp thành công và bạn có thể sử dụng OnCustomer Livechat trên trang web của mình.
